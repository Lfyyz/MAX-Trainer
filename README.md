# MAX Trainer

RU: Программное обеспечение для эффективного заучивания стихотворений.
EN: Software for efficient poetry memorization.

---

## Описание (RU)
MAX Trainer — это инструмент, разработанный для упрощения процесса запоминания текстовой информации. Программа минимизирует когнитивную нагрузку при первичном чтении и постепенно переводит фокус на воспроизведение текста по памяти.

### Основной функционал:
* **Обучение стихотворений**: Специализированный интерфейс для циклического повторения.
* **Постепенное сокращение слов**: Методика поэтапного скрытия текста для тренировки запоминания.
* **Тёмная тема**: Интерфейс с низким уровнем контрастности для снижения нагрузки на зрение.
* **Сохранение текста**: Локальное хранение данных для доступа к загруженным стихам.

---

## Description (EN)
MAX Trainer is a tool designed to simplify the process of memorizing textual information. The software minimizes cognitive load during initial reading and progressively shifts the focus to memory-based recall.

### Key Features:
* **Poetry Learning**: A specialized interface for cyclic repetition and learning.
* **Gradual Word Reduction**: A technique for stepwise text hiding to enhance recall.
* **Dark Theme**: A low-contrast interface designed to reduce eye strain.
* **Text Persistence**: Local data storage for seamless access to uploaded poems.
